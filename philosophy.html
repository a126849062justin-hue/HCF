CODE 01

HONOR

榮譽

倒三角形：穩固根基，直面挑戰。它象徵著我們不屈的意志與穩固的根基。在風暴與挑戰面前，我們如同紮根大地的磐石，將每一次打擊轉化為向下的力量。無論世界如何動盪，我們始終屹立不搖。

CODE 02

COURAGE

勇氣

CODE 03

FAITH

親手重建，絕不認輸。
